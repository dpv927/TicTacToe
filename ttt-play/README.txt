ttt-play - tic-tac-toe play
============================
ttt-play is a fast-responsive tic-tac-toe game made for Linux.

Requirements
------------
In order to compile the applicattion youll need 'make'.

Compilation
-----------
To compile the applicattion, you can use the makefile:
$ make clean all # You can also try 'make clean-build'

Installation
------------
In order to install ttt-search, you can use the implemented makefile:
$ make install

Running dwm
-----------
After the compiling process, you will have a 'ttt-play' executable file at the
/src folder. Then, just follow the instructions at the main README.md file
to pass some arguments to the applicattion.
