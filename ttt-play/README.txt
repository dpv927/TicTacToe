ttt-play - Tic-Tac-Toe Playground
=================================
ttt-play is a fast and responsive Tic-Tac-Toe game designed for Linux.

Requirements
------------
To compile the application, you'll need 'make'.

Compilation
-----------
To compile the application, use the makefile:
$ make clean all # You can also try 'make clean-build'

Installation
------------
To install ttt-play, use the implemented makefile:
$ make install

Running ttt-play
----------------
After the compiling process, you will have a 'ttt-play' executable file in the
/src folder. Then, simply follow the instructions in the main README.md file
to pass arguments to the application.
